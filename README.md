# Googleclone

Este proyecto es una página clone de Google, lo hicimos con HTML, CSS y JavaScript, tiene la funcionalidad de búsqueda que redirige directamente a Google.

Estructura:
GOOGLECLONE/
-- index.html #esta es la página principal
--style.css "aquí están los estilos de la página (centrado, barra, botones)
--script.js #funcionalidad de redirección
--README.md #documentación

Cómo ejecutarlo:
1 Clona el repositorio
2 Abre un servidor local con python: python3 -m http.server 8080
3 En tu navergador abre: http://localhost:8080/index.html
